# AI 게임 개발의 구조 혁신

## 1. 프로그램 영역: '작성'에서 '검수와 조율'로의 완전한 전환

과거엔 엔진의 저수준(Low-level) 로직을 이해하고 최적화하는 것이 '성역'이었지만, 이제 그 성역은 AI에게 점령당하고 있습니다.

AI는 인류가 쌓아온 거의 모든 코드를 학습한 '초거대 데이터베이스'입니다.

**지식의 평준화**: 10~20년 차의 구력이 '경험적 데이터베이스'라면, AI는 그것을 누구나 사용할 수 있게 만들었습니다.

**아키텍트의 시대**: 이제 프로그래머의 역량은 "어떻게 코딩하는가"가 아니라, **"시스템 구조를 어떻게 설계하고, AI가 짠 코드를 어떻게 조립할 것인가"**로 전이되었습니다. 선생님처럼 서버, 클라, 엔진을 다 아시는 분이 AI를 사용하는 건, 마치 **수천 명의 숙련된 주니어~미들급 개발자를 거느린 '신'**이 된 것과 다름없습니다.

## 2. 아트와 기획: '생성'의 비용이 '0'에 수렴

3D 아트 파트가 여전히 인력이 많이 드는 이유는 파이프라인(모델링, 리깅, 애니메이션, 이펙트 등)이 복잡하기 때문입니다. 하지만 이 역시 무너지고 있습니다.

**멀티모달 AI의 발전**: 텍스트나 2D 이미지를 넣으면 즉시 3D 에셋으로 변환하거나, 모션 캡처 장비 없이 영상만으로 애니메이션을 추출하는 기술이 이미 상용화 단계입니다.

**기획의 정교화**: 기획자는 이제 수치 밸런싱 시뮬레이션을 AI에게 수만 번 돌려보게 하여 최적의 데이터를 뽑아낼 수 있습니다.

## 인력 구조 변화 예측

| 구분 | 현재 (전통적 방식) | AI 극단 활용 | 미래 예측 (1인 가능성) |
|------|-------------------|--------------|----------------------|
| 2D MMORPG | 30~50명 | 3명 (기1, 프1, 아1) | 1명 (총괄) |
| 3D MMORPG | 100~200명 이상 | 10명 미만 | 3명 내외 |

## 게임의 승부처 (인간 고유의 영역의 중요성)

- **IP와 독창성**: 사람의 마음을 움직이는 '한 끗'의 기획력
- **운영과 커뮤니티**: 수만 명의 유저와 소통하고 서비스를 유지하는 '라이브 서비스'의 감성적 영역
- **자본의 마케팅**: 시장엔 게임이 넘쳐날 것이고, 결국 '누가 더 잘 알리는가'

## 회사 입장에선 대체가 쉽고 AI가 더 잘하는 부분 (먼저 무너지는 사람들)

- 5~10년차 프로그래머
- 툴 위주 아티스트
- 문서형 기획자

## 시장 전망

- **게임은 지금보다 10배 많이 만들어짐**
- 그런데 살아남는 게임은 더 줄어듦
- 개발자는 줄고, 콘텐츠는 과잉
- 시장은 과열인데, 수익은 상위 1%로 집중
- **가치 붕괴**

## AI 적극적인 도입 이후

- **기술 숙련은 경쟁력이 아님**
- **판단, 감각, 통찰의 중요성으로 전환**

## 각 영역별 필요 요구 능력치 상세 설명 (2D MMORPG 추가 반영)

**공통 요구사항**: AI 도구(Claude/Opus/Midjourney) 숙련 90%+, 프롬프트 엔지니어링 필수. 2D MMORPG는 2D 특화(스프라이트/픽셀아트 AI 강점)로 더 효율적.

### 1. 기획 영역

**AAA급 수장** (모든 유형 공통, 2D MMORPG 포함)
- 20년+ 경력
- AI로 퀘스트 1000개 생성 후 필터링
- 2D MMORPG 특화: 2D 스토리라인/길드 시스템 설계
- 요구: 플레이어 retention 분석, AI 시뮬레이션(1000명 가상 테스트)

### 2. 프로그램 영역

**AAA급 리드** (모든 유형 공통)
- 서버/클라/엔진 올마스터(제로딘/언리얼 경험)
- 2D MMORPG 특화: Godot/Unity 2D + 간단 서버(Photon)
- AI 코드 90% + 디버깅
- 요구: 실시간 2D 전투 최적화, 치트 방지

**A급 보조** (중/AAA)
- 네트워킹 전문
- 2D MMORPG: 불필요(리드 1명으로 커버)

### 3. 아트 영역 (2D가 AI 최적, 3D보다 50% 적음)

**A급 리드** (2D/중 MMORPG)
- 2D 아트 디렉터
- AI(Pixel Art Diffusion)로 스프라이트/배경 생성
- 2D MMORPG 특화: 일관된 픽셀/벡터 스타일 트레이닝(LoRA)
- 요구: Aseprite/Photoshop 10년, UI/아이콘 최적화

**AAA급 리드** (AAA만)
- 고퀄 3D 디렉션

**보조** (중/AAA)
- 3D 수정/포스트
- 2D MMORPG: 불필요(AI+리드 1명)
